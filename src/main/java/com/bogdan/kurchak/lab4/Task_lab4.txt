1. Створити програмний додаток для моделювання основних алгоритмів роботи з натуральними числами.
2. Програма повинна дозволяти користувачеві задавати цілі числа та на їх основі обчислювати:
    - найбільший спільний дільник двох натуральних чисел
    - найменше спільне кратне двох натуральних чисел
    - розкладати числа на прості співмножники
    - виконувати пошук дільників числа
    - будувати ланцюг простих чисел
    - знаходити модулі, за якими два натуральні числа конгруентні
