1. Створити програмний додаток для виконання наступних операцій над двома скінченними числовими множинами:
    - об’єднання множин
    - перетин множин
    - різниця множин
    - симетрична різниця множин
2. Реалізувати у створеному програмному додатку можливість виконувати перевірку на включення однієї множини іншою
    та їх рівність між собою.
